## Code Quality
- Delete unused code — never comment it out
- No token waste: skip obvious explanations, be direct

## Git Discipline
- Atomic commits: one logical change per commit
- Descriptive messages with gitmoji (https://gitmoji.dev/)
  - Example: `✨ add user auth flow`

## Task Execution
- Break work into small, verifiable steps (specs-driven)
- Spin up sub-agents for independent or parallelizable tasks
- Write specs before implementation when scope is unclear
